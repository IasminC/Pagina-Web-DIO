► Headers e parágrafos

Headers no HTML são os cabeçalhos que encontramos no Word ou Google docs, são textos maiores e em negritos que temos pré-programados no HTML nos tamanhos h1, h2, h3, h4, h5 e h6. Sendo o h1 o maior (como encontrado no topo deste site) e h6 o menor. Colocando esses valores entre < > e fechando com < / > (Exemplo: < h1 > Header 1 < /h1>) irá criar o cabeçalho de sua escolha. Após os headers é comum colocarmos algum tipo de texto mas, como fazer isso? No HTML podemos usar a função < p > e, como a maioria das funções em HTML, fechando com < /p > após ter escrito o que desejamos, temos um bloco de texto em nosso site. Como por exemplo, todos os textos neste site estão escrito com blocos de < p >.
Você pode também fazer um < blockquote >, na qual o texto é posto em um começo mais para a direita, demonstrando a ideia de que o que está escrito foi tirado de outro site, logo, é utilizado para referir a citações. Essa função também requere um fechamento (< /blockquote >)
Um exemplo utilizado a definição de < blockquote > utilizando a definição da função pela W3 Schools:

Definition and Usage
The < blockquote > tag specifies a section that is quoted from another source.

Browsers usually indent < blockquote > elements (look at example below to see how to remove the indentation).

► Estilização de texto

Há várias formas de estilização no HTML para o textos:

< b > ou < strong >deixa seu texto em negrito,
< mark > deixa seu texto destacado,
< small > deixa o texto pequeno,
< i > é usado para deixar as palavras em itálico,
< u > irá deixar o texto sublinhado,
< sup > deixará o texto pequeno e suspendido
enquanto o < sub > deixa para baixo,
< del > vai deixa o texto riscado
< abbr title = "" > irá deixar um balão de informação quando colocamos o mouse em cima (Passe o mouse aqui)
e o < hr > vai criar uma linha divisória para seu cabeçalho.


Também existe o < font >, uma função do HTML 4 que não é mais recomendada a utilização já que alguns navegadores não suportam mais e também existem formas mais atualizadas na linguagem CSS. Mas a função agia na atribuição de fonte do texto, tamanho e cor.

Vale lembrar que, como a maioria dos comandos em HTML, tags de estilização precisam ser fechados também, utilizando-as novamente no final mas com uma / antes da tag (Exemplo: < b > < /b >)

► Listas

Listas em HTML são divididas em duas categorias: listas ordenadas e não ordenadas. Utilizamos < ol > para as listas organizadas, onde ficará com uma numeração para cada item.
Exemplo:

  1. Um
  2. Dois
  3. Três
  4. Quatro

Para inserir seus itens na lista, utilizamos < li > e escrevemos o conteúdo que queremos dentro do

. Ficando algo parecido com isto:

< ol >
< li >Um< /li >
< li >Dois< /li >
< li >Três< /li >
< li >Quatro< /li>
< /ol >

Por fim, temos as listas não ordenadas, onde o que diferencia-se da ordenadas é a falta de numeração nos itens, sendo substituído por apenas um círculo. Para fazer uma lista não ordenada, basta utilizar < ul > ao invés de

.

  • Item um
  • Item dois
  • Item três
  • Item quatro

Logo:

< ul >
< li >Item um< /li >
< li >Item dois< /li >
< li >Item três< /li >
< li >Item quatro< /li >
< /ul >

► Links

Links no HTML são geridos pela tag < a >, porém ela chega a ser um pouco mais complicada do que o resto. Não é difícil, mas há apenas alguns detalhes que devem ser notados. Ela deve ser sempre acompanhada, em seu uso, da tag "href" dentro da função com = e "". Visualmente, ficará algo assim:

< a = href="" > Nome do conteúdo < /a >

Dentro das aspas podemos colocar um link que, ao clicar no nome dado para o conteúdo, irá o direcionar para o site programado.

< a = href="https://www.google.com" > Google < /a >

Porém, da forma que está aqui, fará que o link dado abra em cima de nosso site. Uma prática comum no HTML é fazer que os links abram em uma nova aba no navegador. Para isso, temos que adicionar a tag target dentro da tag < a > novamente com "" e =, mas dessa vez colocaremos dentro das aspas o comando "_blank"

< a = href="https://www.google.com" target="_blank"> Google < /a >

Por fim, nosso link aparecerá desta forma:

Google

Há também como você conectar outra página do próprio site com o < a >. Para isso, imagine que temos um outro arquivo HTML de nosso site que está salvo pelo nome de "Sobre-a-empresa" e queremos que o cliente tenha disponível um botão que leve para a descrição da empresa em questão. Para isso, precisamos simplesmente adicionar dentro das aspas de < a = href= "" > o nome do arquivo, que vai estar salvo como .html:

< a = href="sobre-a-empresa.html"> Sobre a empresa < /a >

Podemos também usar essa ancoragem para a mesma página. Como vimos anteriormente, podemos colocar mais informações dentro das tags antes de atribuir como vai aparecer. Podemos também, nesta forma, dar IDs para os comandos usando o comando id = "" dentro da tag usada:

< h1 id="header1" > Header 1 < /h1 >

Aqui, atribuímos dentro do código que o Header 1 terá o ID de "header1". Então, podemos "chamar" essa linha de tag referenciando tal ID. Como fazer isso, então? Podemos referenciar esta linha utilizando href da tag < a >. O que ficaria desta forma:

< a href =" # header1 " >Ir ao Header 1< /a >

Isso transformará o texto em um botão clicável que irá levar a pessoa até onde o "Header 1" está. Um exemplo disso em ação sãos os botões de voltar espalhados por este site e os botões no Índice, no começo da página.
